# Git buster

## Badge
### IF the MR is mine
-Action needed: (default red)
  - [ ] need rebase
  - [ ] unresolved discussions
  - [ ] ci broken
- Wait: (default orange)
  - [ ] need approvals (number in config)
- Done: (default green)
  - [ ] you can merge
### IF the MR not is mine
-Action needed: (default red)
  - [ ] no upvote from me AND not waiting
- Wait: (default orange)
  - [ ] need rebase
  - [ ] unresolved discussions
  - [ ] ci broken
- Done: (default green)
  - [ ] you can merge
